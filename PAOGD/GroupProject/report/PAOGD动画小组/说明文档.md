# 说明文档

### 动画内容简介：

我们的动画内容参考并复现了电影《导火线》中经典的邹兆龙和甄子丹的打斗场景的前面一部分。总时长约30秒，在动画中白色人物模型代表邹兆龙，紫色人物模型代表甄子丹，并简单地搭建了小屋的场景和灯光。

- 人物：邹子龙，甄子丹
- 场景：小屋
- 动画内容：打斗
